# TwistedBot
My telegram bot

尝试用python写个telegram机器人

现阶段实现的功能有：
* /start - 开始使用的引导（暂时不知道写什么）
* /yo - 跟人打个招呼吧
* /cat - 随机发一张从百度图片爬的萌猫照片
* /help - 帮助文档
* echo - 对任何非指令的文本，它会返回相同的内容（学你说话）
